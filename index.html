INSPECT

Intra Node Stencil Performance Evaluation Collection

    INSPECT

    The goal of this website is the collection and presentation of reproducible intranode stencil performance data.

    Automatic generation of loop kernels for several kinds of stencil patterns (see below for configuration details) is done via STEMPEL. In combination with Kerncraft and LIKWID, the INSPECT tool-chain provides a framework to generate and perform static analysis of the stencil code. Data collection can be done with a simple batch script. The presented data allows easy matching of the measured performance data with the automatically generated single- and multicore performance models (ECM and Roofline).

    Machine Descriptions

    Detailed descriptions of the machines used on this website can be found here. Machine files specify information about a machine model for each micro architecture. The Kerncraft repository already contains machine files for the most relevant architectures. Machine files for new hardware can be created with the likwid_bench_auto command provided by Kerncraft.

    Reproducibility

    An important topic with this website is the reproducibility of benchmark results. Every stencil provides information on how the exact benchmarks were run. Together with the machine file specification, which is also linked with every stencil, reproducible benchmark results are possible.

    Find out how to replicate data shown on this website.

    Information on how to perform reproducible performance benchmarks, can be found on this page.
