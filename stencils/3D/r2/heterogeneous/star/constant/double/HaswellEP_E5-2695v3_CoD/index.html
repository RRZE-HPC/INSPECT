    Layer Conditions

    1D Layer Condition:

    • L1: unconditionally fulfilled
    • L2: unconditionally fulfilled
    • L3: unconditionally fulfilled

    2D Layer Condition:

    • L1: P <= 2048/5, that is P ~ 400
    • L2: P <= 16384/5, that is P ~ 3270
    • L3: P <= 1179648/5, that is P ~ 235920

    3D Layer Condition:

    • L1: 32*N*P + 16*P*(N - 2) + 32*P <= 32768, that is N*P ~ 20²
    • L2: 32*N*P + 16*P*(N - 2) + 32*P <= 262144, that is N*P ~ 40²
    • L3: 32*N*P + 16*P*(N - 2) + 32*P <= 18874368, that is N*P ~ 510²

    Have a look at the kernel source code for dimension naming.
