    Layer Conditions

    1D Layer Condition:

    • L1: unconditionally fulfilled
    • L2: unconditionally fulfilled
    • L3: unconditionally fulfilled

    2D Layer Condition:

    • L1: P <= 2050/3, that is P ~ 680
    • L2: P <= 32770/3, that is P ~ 10920
    • L3: P <= 524290/3, that is P ~ 174760

    3D Layer Condition:

    • L1: 16*N*P + 16*P*(N - 1) <= 32768, that is N*P ~ 30²
    • L2: 16*N*P + 16*P*(N - 1) <= 524288, that is N*P ~ 120²
    • L3: 16*N*P + 16*P*(N - 1) <= 8388608, that is N*P ~ 510²

    Have a look at the kernel source code for dimension naming.
